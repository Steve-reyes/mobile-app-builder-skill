# Growth & Viral Loops Template

## 1. Core Growth Model

```
                     ┌──────────────┐
                     │   User Lands │
                     └──────┬───────┘
                            ▼
                     ┌──────────────┐
                     │  Onboards    │
                     └──────┬───────┘
                            ▼
                     ┌──────────────┐
                     │ Core Action  │─── Exit (churn)
                     └──────┬───────┘
                            ▼
                     ┌──────────────┐
                     │  [Viral      │
                     │   Loop]      │─── Invite others → back to top
                     └──────┬───────┘
                            ▼
                     ┌──────────────┐
                     │  Retention   │─── Habit loop
                     └──────┬───────┘
                            ▼
                     ┌──────────────┐
                     │  Monetize    │
                     └──────────────┘
```

## 2. Viral Loop Mechanics

| Loop Type | Trigger | Incentive | Friction Level | Viral Coefficient (K) |
|---|---|---|---|---|
| Referral | "Share with team" | Free month for both | Low | >1.0 if good |
| Content share | User creates something | Exposure | Medium | >0.5 |
| Collaboration | Invite to collaborate | Utility | Low | >1.0 |
| Social proof | "X people use this" | FOMO | Very Low | >0.3 |
| Watermark/branding | Exported content shows logo | Exposure | Zero | Variable |

### Target Viral Coefficient
- **K < 1.0:** Organic growth requires paid acquisition
- **K = 1.0:** Net zero churn — sustainable
- **K > 1.0:** Exponential growth

## 4. Referral Program Design

| Element | Recommendation |
|---|---|
| Reward (referrer) | 1 month free / $10 credit |
| Reward (referred) | 1 month free / $10 credit |
| Trigger | After referred completes core action |
| Friction | One click share via link |
| Tracking | UTM params + unique referral code |
| Cap | Unlimited during launch period |
