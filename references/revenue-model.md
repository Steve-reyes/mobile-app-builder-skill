# Revenue Model & Unit Economics Template

## 1. Revenue Model Options

| Model | Description | Example | Best For |
|---|---|---|---|
| **One-time purchase** | Pay once, use forever | Tarsi, Procreate | Simple tools, privacy-focused |
| **Subscription (monthly/yr)** | Recurring fee | SaaS, Netflix | Ongoing value, cloud services |
| **Freemium** | Free tier + paid upgrade | Spotify, Notion | High virality, low CAC |
| **In-app purchases** | Buy features/consumables | Games, Canva | Add-on value |
| **Usage-based** | Pay per action | Stripe, AWS | Infrastructure, API products |
| **Hybrid** | Base + usage | Slack (free + paid) | Most B2B SaaS |

### LTV Calculation
```
LTV = ARPU × Gross Margin × (1 / Churn Rate)
Example:
ARPU = $15/mo
Gross Margin = 80%
Monthly Churn = 5% (avg lifetime = 20 months)
LTV = $15 × 0.80 × 20 = $240
```

### LTV:CAC Ratio
| Threshold | Verdict |
|---|---|
| < 1:1 | Burning money |
| 3:1 | Healthy SaaS |
| 5:1 | Excellent |
| > 10:1 | Under-investing in growth |

### Payback Period
```
Payback = CAC / (ARPU × Gross Margin)
Target: < 12 months
```
