## 2. Content Funnel

| Stage | Goal | Content Type | Example Title |
|---|---|---|---|
| TOFU (Awareness) | Drive traffic | Blog, video, infographic | "Why [problem] still sucks in 2026" |
| MOFU (Consideration) | Build trust | Case study, comparison, webinar | "[Tool] vs [Competitor]: Which is better?" |
| BOFU (Conversion) | Close | Demo, pricing, testimonial, free trial | "Try [Product] free for 14 days" |

## 4. Backlink Strategy

| Source Type | Examples | Difficulty | Value |
|---|---|---|---|
| Guest posts | Industry blogs, Medium | Medium | High |
| Directory listings | Product Hunt, G2, Capterra | Easy | Medium |
| Resource pages | "Best [category] tools" lists | Hard | High |
| HARO / journalist queries | Help a Reporter Out | Medium | Very High |
| Open source / free tools | Offer free tool in exchange for link | Medium | High |

## 5. Technical SEO

- [ ] Sitemap.xml configured
- [ ] Robots.txt optimized
- [ ] Schema markup (Product, FAQ, HowTo, Article)
- [ ] Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] Mobile-first indexing verified
- [ ] Canonical URLs set
- [ ] Open Graph + Twitter Card meta tags
- [ ] Breadcrumb navigation
- [ ] Pagination (rel=next/prev or infinite scroll SEO)
- [ ] Structured data testing tool verified
